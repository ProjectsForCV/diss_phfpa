Upload a CSV cost matrix in the following format:

n,n,n,n,n,n,n

n,n,n,n,n,n,n

n,n,n,n,n,n,n

Where each row represent an agent and each colum represents a task.

And n is a positive integer that represents an agents preference to the task.

For example:

1,3,5,2,6,2,5,7,8,3,7

2,3,1,4,6,5,8,4,3,8,9

5,1,4,3,6,8,7,9,2,9,9

6,2,3,4,1,7,9,5,8,9,9
