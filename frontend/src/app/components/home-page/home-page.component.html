This application is designed to solve the Assignment Problem using various algorithms.

The linear assignment problem is the problem of choosing the optimal assignment of n agents to m tasks assuming numerical (cost) values are given for each agents preference to each task
The problem can be thought of in terms of a n x n matrix of costs where each row represents an agent and each column represents a task. Cx,y then represents a cost value for a given assignment of agent to task at that position.

Playground

The playground allows you to generate a random cost matrix or upload your own CSV cost matrix.

Here you can play with the various algorithms and view assignments in real time.

Custom Assignment

You may also wish to use the application for real-world scenarios.

In that case you can create a custom assignment which will allow you to survey agents via email and collect preference details.

From here you can view detailed stats about the assignment using various algorithms.
